# ethconnect-swagger-typescript-api

ethconnect-swagger-typescript-api is used to Generate api client from ethconnect's deployed abis.

## Usage
```
Usage: ethconnect-swagger-typescript-api [options]

Options:
  -v, --version                 output the current version
  -u, --base-url <url>          baseURL of ethconnect (default: http://127.0.0.1:8001)
  -o, --outputDir <output>      output directory (default: ./generated)

```
